# OpenLP-CustomStages
Customstage View for OpenLP

This Project inspried by the OpenLP Documentation.
https://manual.openlp.org/stage_view.html#custom-stage-views

Go to the OpenLP Data Folder by going to Tools ‣ Open Data Folder. Inside the data folder create a folder names ‣ stages.
Paste the Above one 

Access:
To access the custom stage view in a browser go to http://myopenlpip:4316/stage/myview.

Eg: http://myopenlpip:4316/stage/songs
